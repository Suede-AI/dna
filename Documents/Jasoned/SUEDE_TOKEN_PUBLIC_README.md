# SUEDE Token

Canonical public token reference for Suede Labs AI (`SUEDE`) and Suede Labs ecosystem context.

Token name: `Suede Labs AI`
Ticker / token: `$SUEDE`

## Overview

`SUEDE` is the utility token connected to Suede Labs, a programmable IP and creator ownership platform for AI-native media.

Suede focuses first on ownership, provenance, rights registration, licensing, monetization, and agent-accessible commerce for creator-owned assets. AI music is a core media use case inside that broader IP layer.

## Ecosystem Positioning

Suede is positioned around:

- Programmable IP
- Creator ownership
- Creative RWA media assets
- Provenance
- Licensing
- x402 payments
- ACP-ready agent commerce
- Multichain and omnichain access
- Base, Ethereum, Avalanche, and Solana

Music is the initial high-signal media category, not the full scope of the project.
